# how to install:

find the config file:

```
cd ~/.config/fastfetch
```

if the config folder for fastfetch doesn't already exist, make one:

```
cd ~/.config

mkdir fastfetch
```
open the file manager and open the config.jsonc
<br>copy and paste my preset:

```
cd ~/.config/fastfetch
```
